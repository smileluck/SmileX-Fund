# 实现贵金属图显示个数设置功能

## 目标

为应用添加设置功能，允许用户配置贵金属图每行显示的个数（1-4个）。

## 实现步骤

### 1. 创建设置组件

* 创建 `app/components/SettingsModal.tsx` 组件

* 实现一个模态框，包含贵金属图每行显示个数的配置选项

* 添加保存按钮，将设置保存到本地存储

### 2. 修改主页面

* 在 `app/page.tsx` 中为设置按钮添加点击事件

* 实现设置模态框的显示/隐藏逻辑

* 添加贵金属图显示个数的状态管理

### 3. 修改 MetalTab 组件

* 修改 `app/components/MetalTab.tsx`，添加 `itemsPerRow` 属性

* 根据 `itemsPerRow` 属性动态调整网格布局

* 确保响应式设计仍然有效

### 4. 本地存储管理

* 使用 `localStorage` 存储用户的设置

* 在应用加载时读取设置

* 确保设置在页面刷新后仍然保持

### 5. 错误处理和边界情况

* 添加输入验证，确保用户只能输入 1-4 之间的数字

* 添加错误处理，确保应用在各种情况下都能正常运行

* 添加默认值，确保即使没有设置也能正常显示

## 技术要点

* 使用 React 的状态管理和副作用钩子

* 使用 Next.js 的组件结构

* 使用 Tailwind CSS 实现响应式布局

* 使用本地存储持久化用户设置

* 实现模态框的动画效果

* 确保代码符合项目的代码风格和规范
